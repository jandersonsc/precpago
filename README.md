# Tecnologias utilizadas
- PHP 8.2
- Nginx
- Laravel 11
- Docker

# Instalação
## Com Docker
### Requisitos
- Docker

Execute no seu terminal:
```
cd $CAMINHO_PROJETO/src
docker-compose up -d
```

## Sem Docker
### Requisitos
- PHP 8.2
- Composer v2

Execute no seu terminal:
```
cd $CAMINHO_PROJETO/src
composer install
php artisan serve
```
